# MATH-METHODS
PROYECTO DE ANALISIS NUMERICO
Este proyecto consiste en el desarrollo de una aplicación interactiva para el análisis y visualización de métodos numéricos de búsqueda de raíces de ecuaciones no lineales. Ejecutar los métodos de Bisección, Newton-Raphson s Iteraciones, mostrando gráficamente el comportamiento iterativo de cada algoritmo.

MÉTODO DE BISECCIÓN:
*Intervalo actual
*Punto medio
*Aproximación de la raíz
*Evolución de las iteraciones

MÉTODO DE NEWTON RAPHSON
*Recta tangente
*Aproximación actual
*Aproximación siguiente
*Convergencia hacia la raíz

TABLA DE ITERACIONES:
Iteración
Aproximación
Valor de la función
Error absoluto
Error relativo
